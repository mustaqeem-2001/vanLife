
## About

Van Life is a React web app for renting and managing travel vans. Users can browse vans with filters, view details, and book trips, while hosts can track income, manage vans, and monitor reviews. Built with React, React Router, and Firebase, it demonstrates modern front-end architecture, dynamic data handling, and secure authentication. This website is designed for small screen users.

## Key Features

Public Van Listings: Browse vans by type (Simple, Luxury, Rugged) with dynamic filtering and detailed individual van pages.

Authentication: Secure login system protecting host-specific routes, with seamless redirection for unauthorized users.

Host Dashboard: Complete management panel for van owners including income tracking, reviews, and van management.

CRUD Capabilities for Hosts: Hosts can view van details, update pricing and photos, and monitor performance metrics.

Data Management: Firebase Firestore handles dynamic data fetching for vans, hosts, and reviews with optimized queries.

## Technologies

Frontend: React, React Router, CSS
Backend: Firebase Firestore
Tools: Fetch API for authentication, React Context for state sharing, React Icons for UI enhancements
